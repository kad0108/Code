**题意：给一个数n，计算所有<=n的数中1的个数**

**样例：**
```
Given n = 13,
Return 6, because digit 1 occurred in the following numbers: 1, 10, 11, 12, 13.
```

<br/>
**循环叠加每一位上是1的个数，每一位上是1的个数的计算分三种情况，如下**
```
以百位h上为1为例：
h = 0时： n = 12013, a = 120, b = 13, 
	百位为1的情况为00100～00199,01100~01199...11100~11199,
	一共 a/10 * 100 = 1200 个

h = 1时： n = 12113, a = 121, b = 13,
	百位为1的情况为00100~00199,01100~01199...11100~11199,12100~12113,
	一共 a/10 * 100 + (b+1) = 1200 + 14 = 1214 个

h > 1时： n = 12413, a = 124, b = 13,
	百位为1的情况为00100~00199,01100~01199...11100~11199,12100~12199,
	一共 (a/10 + 1) * 100 = 1300 个
```

```
class Solution {
public:
    int countDigitOne(int n) {
        int ans = 0;
        for(long long i = 1; i <= n; i*=10)//防止爆int
        {
            int a = n/i, b = n%i;
            int digit = a % 10;
            if(digit == 0) ans += (a/10 * i);
            else if(digit == 1) ans += (a/10 * i + b + 1);
            else if(digit > 1) ans += (a/10 + 1)*i;
        }
        return ans;
    }
};
```
