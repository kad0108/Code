> 从1到n整数中1出现的次数。比如n＝12，1～12中包含1的数字有1、10、11、12共出现了5次。

**分析：O(logn)**

```
以n的百位数字h为例：

h=0: n=12034，以百位分割，高位a＝120，低位b＝34
	百位为1的情况有：00100～00199、01100～01199...09100～09199、10100~10199、11100～11199
	则共有a/10*100 = 12*100个

h=1: n=12134，a=121, b=34
	百位为1的情况有：00100～00199...11100～11199、12100～12134
	共有a/10*100+(b+1) = 12*100+(34+1)个

h>1: n=12634, a=126, b=34
	百位为1的情况有：00100~00199...11100~11199、12100~12199
	共有(a/10+1)*100 = 13*100个
```

```c++
int NumberOf1Between1AndN(int n)
{
	int ans = 0;
    for(long long i = 1; i <= n; i *= 10){
        int high = n / i;
        int low = n % i;
        int digit = high % 10;
        if(digit == 0) ans += high/10*i;
        else if(digit == 1) ans += (high/10*i + (low+1));
        else ans += (high/10+1)*i;
    }
    return ans;
}
```
