> 题目描述

> 输入一个整数，输出该数二进制表示中1的个数。其中负数用补码表示。

**特殊情况：n为负数，负数右移左补1**

```c++
class Solution {
public:
	/** 两种写法
     int NumberOf1(int n) {
         int ans = 0;
         int cot = 1;
         while(n){
             if(cot > 32) break;
             if(n & 1) ans++;
             n >>= 1;
             cot++;
         }
         return ans;
     }
     */
     int NumberOf1(int n){
     	int ans = 0;
     	unsigned int flag = 1;
     	while(flag){
     		if(n & flag) ans++;
     		flag <<= 1;
     	}
     	return ans;
     }
};
```

**惊喜解法：```n & (n-1)```会消去n最低位的1，比如```n=1100,n-1=1011,n&(n-1)=1000```**
```c++
int NumberOf1(int n){
 	int ans = 0;
 	while(n)
 	{
 		n = n & (n-1);
 		ans++;
 	}
 	return ans;
 }
```

### 相关题目


**判断一个整数是不是2的整数次方。**

**如果一个整数是2的整数次方，那么二进制表示一定只有一个1。那么n&(n-1)就会消去唯一的1返回0。**

```c++
bool isPowerofTwo(int n)
{
	return (n > 0) && (n & (n-1)) == 0;
}
```
